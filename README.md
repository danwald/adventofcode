# Advent of code

Repository of submissions to [Advent of Code](https://adventofcode.com/) (by year)

## Table of Contents

- [Usage](#usage)
- [License](#license)

## Usage

**`python>=3.8+`**
- stdin terminates on empties
- will execute 2nd solution if applicable

```bash
python <YYYY>/dayDD_<puzzlename>.py < stdin
```
